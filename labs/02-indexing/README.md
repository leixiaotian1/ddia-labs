# 索引结构

## 概述

本lab实现了两种基本的索引结构，展示它们的特点和适用场景：
- 哈希索引（Hash Index）
- B-tree索引（B-tree Index）

## 索引类型对比

### 哈希索引
- **特点**: O(1)查找，无序
- **优势**: 点查询极快
- **劣势**: 不支持范围查询，哈希冲突处理

### B-tree索引
- **特点**: O(log n)查找，有序
- **优势**: 支持范围查询，数据有序
- **劣势**: 查找速度略慢于哈希索引

## 运行方式


## 关键权衡

- **查找速度**: 哈希索引 O(1) vs B-tree索引 O(log n)
- **范围查询**: 哈希索引不支持，B-tree索引支持
- **有序性**: 哈希索引无序，B-tree索引有序
- **适用场景**:
  - 哈希索引: 等值查询，不需要范围查询
  - B-tree索引: 需要范围查询或有序遍历
