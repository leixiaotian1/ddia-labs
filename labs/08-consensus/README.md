# 共识算法

## 概述

本lab实现了分布式系统中的共识算法：
- Raft算法（重点实现）
- Paxos简化版

## 对应DDIA章节

- 第9章：一致性与共识
- 第9.4节：分布式事务与共识

## 共识算法

### Raft算法
- **特点**: 易于理解的共识算法，通过选主、日志复制实现共识
- **优势**: 易于理解和实现
- **应用**: etcd, Consul等

### Paxos算法
- **特点**: 经典的共识算法，理论完备
- **优势**: 理论证明完备
- **劣势**: 实现复杂，难以理解

## 运行方式

```bash
# 运行Raft算法示例
cd raft
go run main.go

# 运行Paxos简化版示例
cd paxos-simple
go run main.go
```

## 关键权衡

- **理解性**: Raft比Paxos更容易理解
- **性能**: 两者性能相近
- **实现复杂度**: Raft实现相对简单
- **适用场景**: 需要强一致性的分布式系统
