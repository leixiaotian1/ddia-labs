# 时钟同步问题

## 概述

本lab演示了分布式系统中的时钟同步问题，以及逻辑时钟的解决方案。

## 时钟问题

分布式系统中，物理时钟可能不同步，导致事件排序错误。逻辑时钟提供了一种不依赖物理时钟的解决方案。

## 运行方式

```bash
cd logical-clock
go run main.go
```

## 关键概念

- **物理时钟**: 依赖系统时钟，可能不同步
- **逻辑时钟**: 基于事件顺序，不依赖物理时钟
- **Lamport时钟**: 简单的逻辑时钟实现
