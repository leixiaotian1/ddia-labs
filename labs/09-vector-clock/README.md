# 向量时钟

## 概述

本lab演示了向量时钟（Vector Clock）在分布式系统中用于事件排序的应用。

## 对应DDIA章节

- 第9章：一致性与共识
- 第9.4节：分布式事务与共识

## 向量时钟

向量时钟用于在分布式系统中确定事件的因果关系（happens-before关系），比逻辑时钟更强大。

## 运行方式

```bash
cd event-ordering
go run main.go
```

## 关键概念

- **事件排序**: 确定分布式系统中事件的先后顺序
- **因果关系**: 判断两个事件是否存在因果关系
- **并发事件**: 识别没有因果关系的事件（并发事件）
